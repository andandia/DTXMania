# DTXMania クローン / DTXビューア アプリの概要

このアプリケーションは、DTXファイルを読み込んで音楽ゲームの譜面を再生する、DTXManiaのクローンまたはビューアです。MIDIコントローラーでの入力や、対応する音声・画像メディアファイルの読み込みに対応しています。

## 動作環境・開発環境

- **ゲームエンジン**: Unity 2020.3.24f1
- **主要な使用言語**: C#

## 主な機能と構成

- **DTXファイルのパース**: `DTXHelper.cs` や `DTXInputOutput.cs` を使用して、DTXファイル（楽曲情報、チップリスト、BPM、画像・動画などのパスを含む独自形式）を読み込みます。ファイルの構造に関する詳細は `DTXFileStructure.md` を参照してください。
- **ゲームの進行・管理**: `SongManager.cs` や `StageManager.cs` などで、ロードされた楽曲情報の管理やステージ（ゲーム画面）の進行を行います。
- **入力・デバイスの処理**: `InputDevice.cs` や `MIDIControllerTest.cs` などから、MIDI入力やその他のキー入力デバイスを処理し、リズムゲームとしての入力を受け付けます。
- **音声の再生**: `SoundManager.cs` や Unityの `AudioManager` 設定を通して、チップの音声ファイルを再生します。

## 使用方法（概要）

1. アプリケーションにDTXファイル（および付随するWAV/OGG等の音声、画像ファイル）が含まれるフォルダを読み込ませます。
2. アプリケーションがDTXファイルを解析し、譜面（チップ）やメディアをロードします。
3. Unityのシーン（StageManagerなど）を通じて、ロードされた譜面を再生・プレイすることができます。
4. MIDIコントローラーなどのデバイスを使用し、落ちてくるチップのタイミングに合わせて入力を行います。
